# x-reader - 深度解读

## 1. 核心洞察 (Insight)

**核心观点**：x-reader 是通用内容摄入系统——URL 到结构化内容的跨平台解决方案。

**关键洞见**：
- **三层架构**：Python CLI → Claude-Code Skills → MCP Server
- **中文优先**：微信、小红书、B站、小宇宙的深度支持
- **Whisper 转录**：视频/播客的完整语音转文字

> "Universal content reader — fetch, transcribe, and digest content from any platform."

## 2. 架构分析 (Architecture)

### 三层架构
| 层级 | 功能 | 格式 |
|------|------|------|
| **Layer 1** | Python CLI/Library | 基础内容获取 |
| **Layer 2** | Claude-Code Skills | 视频转录 + AI 分析 |
| **Layer 3** | MCP Server | 工具化集成 |

### 平台支持矩阵
| 平台 | 文本 | 视频/音频 | 特殊说明 |
|------|------|-----------|----------|
| YouTube | Jina | yt-dlp + Whisper | Groq API |
| Bilibili | API | Claude-Code skill | - |
| X/Twitter | Jina → Playwright | - | - |
| WeChat | Jina → Playwright | - | 公众号 |
| Xiaohongshu | Jina → Playwright | - | 需登录 |
| Telegram | Telethon | - | - |
| RSS | feedparser | - | - |
| 小宇宙 | - | Claude-Code skill | 播客 |
| Apple Podcasts | - | Claude-Code skill | - |
| 任意网页 | Jina | - | 通用 |

### 架构流程
```
User sends URL
    │
    ├─ Text content (article, tweet, WeChat)
    │   └─ Python fetcher → UnifiedContent → inbox
    │
    ├─ Video (YouTube, Bilibili, X video)
    │   ├─ Python fetcher → metadata
    │   └─ Video skill → full transcript via subtitles/Whisper
    │
    ├─ Podcast (小宇宙, Apple Podcasts)
    │   └─ Video skill → full transcript via Whisper
    │
    └─ Analysis requested
        └─ Analyzer skill → structured report + action items
```

## 3. 关键方法论 (Methodology)

### 反爬策略
| 平台 | 策略 |
|------|------|
| 微信/小红书 | Jina Reader → Playwright 备用 |
| 小红书 | 需一次性登录保存 session |

### Whisper 转录流程
```
视频 URL
    ↓
yt-dlp 提取字幕（如有）
    ↓
无字幕 → Groq Whisper API 转录
    ↓
完整文本 + 时间戳
```

### MCP 工具暴露
- `read_url(url)` — 获取任意 URL
- `read_batch(urls)` — 批量获取
- `list_inbox()` — 查看历史
- `detect_platform(url)` — 平台识别

## 6. 批判性思考 (Critique)

### 优点
- 中文平台覆盖最全
- 三层架构灵活（CLI/Skill/MCP）
- Whisper 转录高质量
- Obsidian 集成开箱即用

### 局限
- 复杂度较高（多层依赖）
- Groq API 有速率限制
- Playwright 首次安装较重
- 小红书等平台需要登录

### 与 DeepReader 对比
| 维度 | x-reader | DeepReader |
|------|----------|-----------|
| 定位 | 通用 + 中文 | OpenClaw 专用 |
| 复杂度 | 高 | 低 |
| 视频转录 | Whisper（高质量） | 字幕（快速） |
| 中文平台 | 深度支持 | 通用支持 |
| 使用门槛 | 需配置 | 零配置 |
| MCP | 支持 | 不支持 |

### 适用场景
- **x-reader**：中文内容创作者、播客转录、复杂工作流
- **DeepReader**：OpenClaw 用户、快速阅读、英文内容为主
