# anthropics/knowledge-work-plugins: Claude Cowork зҹҘиҜҶе·ҘдҪңжҸ’д»¶

## дёҖеҸҘиҜқжҰӮиҝ°

Knowledge Work Plugins жҳҜ Anthropic ејҖжәҗзҡ„ 11 дёӘ Claude Cowork жҸ’д»¶йӣҶеҗҲпјҢе°Ҷ Claude иҪ¬еҸҳдёәй”Җе”®гҖҒе®ўжҲ·ж”ҜжҢҒгҖҒдә§е“Ғз®ЎзҗҶгҖҒиҗҘй”ҖгҖҒжі•еҫӢгҖҒиҙўеҠЎзӯүи§’иүІзҡ„дё“дёҡеҠ©жүӢпјҢйҖҡиҝҮ SkillsгҖҒConnectorsгҖҒCommands е®һзҺ°еҚіиЈ…еҚіз”Ёзҡ„дё“дёҡе·ҘдҪңжөҒгҖӮ

## йЎ№зӣ®е®ҡдҪҚ

**зӣ®ж Үз”ЁжҲ·**:
- дҪҝз”Ё Claude Cowork зҡ„зҹҘиҜҶе·ҘдҪңиҖ…
- йңҖиҰҒи§’иүІдё“дёҡеҢ–иғҪеҠӣзҡ„еӣўйҳҹ
- еёҢжңӣе®ҡеҲ¶ Claude е·ҘдҪңжөҒзҡ„з»„з»Ү

**и§ЈеҶізҡ„й—®йўҳ**:
- **Claude йҖҡз”ЁжҖ§иҝҮејә**: зјәд№Ҹзү№е®ҡи§’иүІзҡ„дё“дёҡе·ҘдҪңжөҒжҢҮеҜј
- **е·Ҙе…·йӣҶжҲҗеҲҶж•Ј**: йңҖиҰҒдёәжҜҸдёӘе·Ҙе…·еҚ•зӢ¬й…ҚзҪ®иҝһжҺҘ
- **еӣўйҳҹеҚҸдҪңдёҚдёҖиҮҙ**: дёҚеҗҢжҲҗе‘ҳдҪҝз”Ё Claude ж–№ејҸдёҚз»ҹдёҖ
- **е…¬еҸёжөҒзЁӢйҡҫдј йҖ’**: еҶ…йғЁжңҜиҜӯгҖҒжөҒзЁӢйҡҫд»Ҙзј–з ҒеҲ° AI еҠ©жүӢдёӯ

**дҪҝз”ЁеңәжҷҜ**:
- й”Җе”®еӣўйҳҹз ”з©¶жҪңеңЁе®ўжҲ·гҖҒеҮҶеӨҮйҖҡиҜқ
- е®ўжҲ·ж”ҜжҢҒеӨ„зҗҶе·ҘеҚ•гҖҒж’°еҶҷзҹҘиҜҶеә“
- дә§е“Ғз®ЎзҗҶеҶҷ PRDгҖҒи§„еҲ’и·Ҝзәҝеӣҫ
- иҗҘй”ҖеӣўйҳҹеҲӣдҪңеҶ…е®№гҖҒз®ЎзҗҶе“ҒзүҢ
- жі•еҫӢеӣўйҳҹе®Ўж ёеҗҲеҗҢгҖҒеӨ„зҗҶ NDAs
- иҙўеҠЎеӣўйҳҹзј–еҲ¶жҠҘиЎЁгҖҒж”ҜжҢҒе®Ўи®Ў

**дёҺеҗҢзұ»йЎ№зӣ®е·®ејӮ**:
- **Anthropic е®ҳж–№еҮәе“Ғ**: 11 дёӘжҸ’д»¶пјҢиҰҶзӣ–дё»иҰҒзҹҘиҜҶе·ҘдҪңеІ—дҪҚ
- **еҚіиЈ…еҚіз”Ё**: Skills иҮӘеҠЁи§ҰеҸ‘пјҢCommands жҳҫејҸи°ғз”Ё
- **еҸҜе®ҡеҲ¶**: дҝ®ж”№ connectorsгҖҒж·»еҠ е…¬еҸёдёҠдёӢж–ҮгҖҒи°ғж•ҙе·ҘдҪңжөҒ
- **ж–Үд»¶й©ұеҠЁ**: Markdown + JSONпјҢж— д»Јз ҒгҖҒж— еҹәзЎҖи®ҫж–Ҫ
- **еҸҢе№іеҸ°ж”ҜжҢҒ**: Claude Cowork + Claude Code

## README дёӯж–Үз®Җд»Ӣ

**Knowledge Work Plugins** - е°Ҷ Claude иҪ¬еҸҳдёәдҪ зҡ„и§’иүІгҖҒеӣўйҳҹгҖҒе…¬еҸёзҡ„дё“дёҡеҠ©жүӢгҖӮ

**дёәд»Җд№ҲйңҖиҰҒжҸ’д»¶**:
Cowork и®©дҪ и®ҫе®ҡзӣ®ж ҮпјҢClaude дәӨд»ҳе®ҢжҲҗзҡ„дё“дёҡе·ҘдҪңгҖӮжҸ’д»¶и®©дҪ жӣҙиҝӣдёҖжӯҘпјҡе‘ҠиҜү Claude дҪ е–ңж¬ўзҡ„е·ҘдҪңж–№ејҸгҖҒдҪҝз”Ёе“Әдәӣе·Ҙе…·е’Ңж•°жҚ®гҖҒеҰӮдҪ•еӨ„зҗҶе…ій”®е·ҘдҪңжөҒгҖҒжҡҙйңІе“Әдәӣж–ңжқ е‘Ҫд»Ө вҖ”вҖ” и®©дҪ зҡ„еӣўйҳҹиҺ·еҫ—жӣҙеҘҪгҖҒжӣҙдёҖиҮҙзҡ„з»“жһңгҖӮ

**жҸ’д»¶з»„жҲҗ**:
жҜҸдёӘжҸ’д»¶жү“еҢ…дәҶпјҡ
