# tw93/Kaku: AI 编码终端

## 一句话概述

Kaku 是一个专为 AI 编码设计的快速开箱即用终端，是 WezTerm 的深度定制分支，提供零配置体验、macOS 原生字体渲染、主题感知和 AI 面板集成。

## 项目定位

**目标用户**:
- 寻求更好终端体验的开发者
- 需要 AI 辅助编码的工程师
- 喜欢 JetBrains Mono 字体的用户
- 追求开箱即用体验的 macOS 用户

**解决的问题**:
- **配置复杂**: 传统终端需要大量配置才能达到理想效果
- **字体渲染差**: 默认字体和渲染不够理想
- **主题切换麻烦**: 需要手动切换明暗主题
- **AI 集成缺失**: 终端缺乏 AI 辅助功能
- **启动慢**: 终端启动和响应速度慢

**使用场景**:
- 日常开发终端操作
- AI 辅助编码工作流
- 多窗口/多面板终端管理
- 跨平台一致终端体验

**与同类项目差异**:
- **零配置**: 默认 JetBrains Mono、macOS 字体渲染、低分辨率字体适配
- **主题感知**: 自动跟随 macOS 明暗主题切换
- **AI 面板**: 内置 AI 辅助面板 (Cmd+Shift+A)
- **轻量快速**: 40% 更小二进制，即时启动，懒加载
- **WezTerm 兼容**: 完整兼容 WezTerm Lua 配置

## 运行与开发方式

**安装**:
```bash
# Homebrew
brew install tw93/tap/kakuku

# 或下载 DMG 手动安装
```

**开发**:
```bash
# 克隆仓库
git clone https://github.com/tw93/Kaku.git
cd Kaku

# 构建 (需要 Rust 工具链)
cargo build --release

# 运行
./target/release/kaku
```

**快捷键**:
- Cmd+T: 新建标签
- Cmd+N: 新建窗口
- Cmd+W: 关闭标签/面板
- Cmd+D: 垂直分割
- Cmd+Shift+D: 水平分割
- Cmd+Shift+A: AI 面板
- Cmd+,: 设置面板

## 外部接口

**配置格式**:
WezTerm 兼容的 Lua 配置:
```lua
-- ~/.config/kaku/kaku.lua
local wezterm = require 'wezterm'
local config = {}

-- 字体配置
config.font = wezterm.font('JetBrains Mono')

-- 主题配置
config.color_scheme = 'Dark+'

return config
```

**AI 面板**:
- 快捷键: Cmd+Shift+A
- 功能: AI 辅助编码集成
