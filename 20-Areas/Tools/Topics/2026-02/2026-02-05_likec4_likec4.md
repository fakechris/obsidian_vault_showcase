# likec4/likec4: 架构即代码可视化工具

## 一句话概述

LikeC4 是一个受 C4 模型和 Structurizr DSL 启发的架构建模语言及工具，允许开发者用代码描述软件架构，生成始终保持最新的实时架构图，支持自定义符号和多层嵌套架构。

## 项目定位

**目标用户**:
- 软件架构师和系统设计人员
- 技术文档编写者
- 需要维护架构文档的开发团队
- 追求文档即代码实践的工程团队

**解决的问题**:
- **架构图过时**: 传统架构图与代码不同步，很快过时
- **工具碎片化**: 使用不同工具绘制架构图，难以统一
- **协作困难**: 架构图版本控制困难，协作效率低
- **定制性不足**: 现有工具难以自定义符号和风格

**使用场景**:
- 从代码生成实时架构图
- 团队协作维护架构文档
- 多层架构可视化(系统→容器→组件→代码)
- 自定义架构符号和风格

**与同类项目差异**:
- **灵感来源**: 受 C4 模型和 Structurizr DSL 启发
- **灵活性**: 支持自定义符号、元素类型和多层嵌套
- **实时更新**: 架构图始终与代码保持同步
- **开源免费**: MIT 许可证，社区驱动
- **TypeScript**: 现代 TypeScript 代码库

## README 中文简介

**LikeC4** - 架构即代码: 可视化、协作、演进你的软件架构

**什么是 LikeC4?**

LikeC4 是一个用于描述软件架构的建模语言，以及从模型生成图表的工具。灵感来自 C4 模型和 Structurizr DSL，但提供更多灵活性：

- 自定义或定义你自己的符号
- 自定义元素类型
- 任意数量的嵌套架构层次
- 完美适配你的需求

**示例代码**:

```likec4
specification {
  element actor {
    style {
      shape person
      color secondary
    }
  }
  element system
  element container
}

model {
  user = actor 'User' {
    description 'A user of the system'
  }

  saas = system 'Our SaaS' {
    container api 'API' {
      description 'REST API'
      technology 'Node.js'
    }
    container web 'Web App' {
      description 'React application'
      technology 'React'
    }
  }

  user -> api 'uses'
  web -> api 'calls'
}

views {
  view index {
    title 'System Overview'
    include *
  }
}
```

**快速开始**:

```bash
# 安装 CLI
npm install -g likec4

# 启动预览
npx likec4 start

# 或安装依赖后
pnpm install
pnpm dev
```

**功能特点**:
- **DSL 语言**: 专门设计的架构描述语言
- **实时预览**: 修改代码即时更新图表
- **多层级**: 支持 C4 模型的四个层级
- **自定义符号**: 定义自己的元素样式和形状
- **导出**: 支持多种格式导出图表

## 运行与开发方式

**开发设置**:
```bash
# 克隆仓库
git clone https://github.com/likec4/likec4.git
cd likec4

# 安装依赖 (使用 pnpm)
pnpm install

# 启动开发
pnpm dev

# 构建
pnpm build
```

**基本使用**:
```bash
# 全局安装
npm install -g likec4

# 启动预览服务器
npx likec4 start

# 构建图表
npx likec4 build

# 导出图表
npx likec4 export
```

**在项目中使用**:
1. 创建 `.likec4` 文件定义架构
2. 运行 `npx likec4 start` 预览
3. 使用 React 组件嵌入文档

## 外部接口

**CLI 命令**:
| 命令 | 功能 |
|------|------|
| `likec4 start` | 启动开发预览服务器 |
| `likec4 build` | 构建架构图 |
| `likec4 export` | 导出为各种格式 |
| `likec4 check` | 验证模型 |

**React 组件**:
```jsx
import { LikeC4View } from '@likec4/react'

function Documentation() {
  return <LikeC4View viewId="index" />
}
```

**语言特性**:
- `specification`: 定义元素类型和样式
- `model`: 定义架构模型和关系
- `views`: 定义图表视图
- `include`: 包含元素到视图
