# robinshi2010/AItoType: 开源语音转文字工具

## 一句话概述

AItoType 是一个基于 Tauri v2 构建的开源免费桌面端语音转文字工具，支持全局快捷键唤醒、自动粘贴到当前输入框、历史记录管理，支持 OpenRouter 和 SiliconFlow API。

## 项目定位

**目标用户**:
- 需要语音输入提高效率的用户
- 寻求免费语音转文字方案的开发者
- 喜欢开源桌面工具的用户
- 需要多语言语音识别的用户

**解决的问题**:
- **输入效率低**: 打字速度慢，语音输入更快
- **商业软件贵**: 语音输入软件收费高
- **隐私担忧**: 商业软件数据处理方式不透明
- **跨应用粘贴**: 语音结果需要手动复制粘贴
- **API 选择少**: 缺乏灵活的 API 提供商选择

**使用场景**:
- 快速语音输入到聊天软件
- 语音转文字记录笔记
- 多语言语音输入
- 无障碍辅助输入

**与同类项目差异**:
- **完全开源免费**: 严禁商业盈利行为
- **自动粘贴**: 识别结果自动写入当前光标位置
- **后台运行**: 支持关闭主窗口后台运行，快捷键唤醒
- **灵动胶囊**: 后台录音时屏幕底部显示状态条
- **多供应商**: 支持 OpenRouter 和 SiliconFlow

## README 中文简介

**AItoType** - 开源、免费的桌面端语音转文字工具

基于 Tauri v2 构建，旨在提供极致轻量、高效的语音输入体验。

**核心理念**: 免费开源、极简设计、高效生产力

**⚠️ 声明**: 本项目完全开源免费，供大家学习与使用。严禁用于任何商业盈利行为。

**快速指南**:

**首次配置**:
1. 首次打开需进入配置界面
2. 两步完成: (1) 选择供应商 (2) 填入 API Key

**界面交互**:
- 点击主界面"录制"按钮开始，再次点击停止
- 识别结果会显示在主界面并可一键复制

**后台/快捷键模式**:
- 支持关闭主窗口后台运行
- 通过预设快捷键唤醒录制，再次按键停止
- 识别完成后会自动写入当前光标所在输入框（自动粘贴）

**历史记录**:
- History 页面按条目卡片展示
- 每条可点击直接复制内容

**权限引导**:
- 首次使用会引导完成麦克风与辅助功能授权
- 确保可录音和后台自动写入

**供应商**:
- 目前支持 OpenRouter 和 SiliconFlow
- OpenRouter 默认使用 Gemini 3.0 Flash
- SiliconFlow 默认使用 TeleAI 模型
- 划重点: 硅基流动的 TeleAI/TeleSpeechASR 完全免费，速度快且随便用

**功能特性**:

**全局快捷键唤起**:
- 默认 Alt+Space（Windows 默认 Ctrl+Shift+Space）
- 一键录音，再次按下停止并识别

**无缝嵌入工作流**:
- 自动写入（自动粘贴）: 识别完成后结果自动"打字"到当前光标所在输入框
- 自动复制: 可选自动复制结果到剪贴板

**历史记录优化**:
- 历史页面保持大面板展示
- 每条记录使用独立小卡片显示时间与文本
- 点击任意历史卡片可立即复制该条内容

**极致 UI 设计**:
- "Glass Monolith" 玻璃拟态设计风格
- 沉浸式动效与交互反馈
- 后台悬浮条: 后台录音时屏幕底部显示灵动胶囊状态条，不干扰视线

**权限引导更清晰**:
- 启动后按提示完成系统权限授权（麦克风、辅助功能）
- 未授权时会给出可操作提示

**灵活的模型支持**:
- 内置 OpenRouter 支持
- 默认 Gemini 3.0 Flash（速度快、精度高、免费额度充裕）
- 可自定义 API Key 和 Model（如 GPT-4o, Claude 3.5 Sonnet 等）

**隐私安全**:
- 音频数据仅在您的设备上录制
- 直接发送至您配置的 API 服务商
- 本地不留存录音音频文件
- 保存必要配置（API Key、Model、快捷键）以便下次使用

**技术栈**:
- **Core**: Rust (Tauri v2, cpal, hound, arboard, enigo)
- **Frontend**: Vanilla JS + CSS (无框架，追求极致轻量与性能)
- **API**: OpenRouter / SiliconFlow (兼容 OpenAI 格式)

## 运行与开发方式

**快速开始**:
1. 下载并安装 AItoType
2. 首次配置：选择供应商 + 填入 API Key
3. 点击"录制"开始语音输入
4. 或使用快捷键 Alt+Space 唤醒

**开发**:
```bash
# 克隆仓库
git clone https://github.com/robinshi2010/AItoType.git
cd AItoType

# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建
npm run tauri build
```

**API 配置**:
- OpenRouter: 默认 Gemini 3.0 Flash
- SiliconFlow: 默认 TeleAI（完全免费）
- 可自定义 API Key 和 Model
