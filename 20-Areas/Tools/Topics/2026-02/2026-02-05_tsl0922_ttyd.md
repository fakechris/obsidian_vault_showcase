# tsl0922/ttyd: Web 终端共享工具

## 一句话概述

ttyd 是一个简单的命令行工具，用于通过网络共享终端，基于 libuv 和 WebGL2 构建，支持 CJK、IME、ZMODEM 文件传输、Sixel 图像输出和 SSL。

## 项目定位

**目标用户**:
- 需要远程访问终端的开发者
- 需要在浏览器中使用终端的用户
- 嵌入式设备管理者
- 需要轻量级远程 shell 的管理员

**解决的问题**:
- **SSH 限制**: 某些环境无法使用 SSH
- **客户端安装**: 需要预装客户端才能远程访问
- **Web 访问需求**: 希望通过浏览器访问终端
- **移动设备访问**: 手机平板难以使用传统终端
- **文件传输**: 需要简单的文件传输方式

**使用场景**:
- 嵌入式设备 Web 终端
- 容器 Web 控制台
- 远程服务器管理
- 在线编程环境
- 快速共享终端会话

**与同类项目差异**:
- **C 语言编写**: 轻量高效，低资源占用
- **libuv + WebGL2**: 高性能，流畅体验
- **全功能终端**: CJK、IME、ZMODEM、Sixel 支持
- **SSL 支持**: OpenSSL/Mbed TLS 加密
- **跨平台**: macOS、Linux、BSD、OpenWrt、Windows

## README 中文简介

**ttyd** - 通过网络共享您的终端

ttyd 是一个简单的命令行工具，用于通过网络共享终端。

**特性**:

- **基于 libuv 和 WebGL2**: 为速度而构建
- **全功能终端**: CJK 和 IME 支持
- **ZMODEM (lrzsz) / trzsz 文件传输**: 支持文件传输
- **Sixel 图像输出**: 支持 img2sixel / lsix
- **SSL 支持**: 基于 OpenSSL / Mbed TLS
- **自定义命令**: 运行任何带选项的自定义命令
- **基本认证**: 支持和许多其他自定义选项
- **跨平台**: macOS、Linux、FreeBSD/OpenBSD、OpenWrt、Windows

**安装**:

**macOS**:

**Homebrew**:
```bash
brew install ttyd
```

**MacPorts**:
```bash
sudo port install ttyd
```

**Linux**:

各发行版包管理器安装方式参见文档。

**使用**:

```bash
# 共享当前 shell
ttyd bash

# 共享指定命令
ttyd -p 8080 python

# 带 SSL
ttyd --ssl --ssl-cert server.crt --ssl-key server.key bash

# 带认证
ttyd -c username:password bash
```

## 运行与开发方式

**安装**:

**macOS**:
```bash
brew install ttyd
# 或
sudo port install ttyd
```

**Linux**:
```bash
# Ubuntu/Debian
sudo apt install ttyd

# Fedora
sudo dnf install ttyd

# Arch
sudo pacman -S ttyd
```

**从源码**:
```bash
git clone https://github.com/tsl0922/ttyd.git
cd ttyd
mkdir build && cd build
cmake ..
make
sudo make install
```

**使用**:

```bash
# 基本用法
ttyd bash

# 指定端口
ttyd -p 8080 bash

# SSL
ttyd --ssl --ssl-cert cert.pem --ssl-key key.pem bash

# 基本认证
ttyd -c user:pass bash

# 自定义命令
ttyd -p 8080 python3

# Docker 中运行
docker run -it --rm -p 7681:7681 tsl0922/ttyd bash
```

## 外部接口

**CLI 选项**:
| 选项 | 说明 |
|------|------|
| `-p, --port` | 监听端口 (默认 7681) |
| `-i, --interface` | 绑定接口 |
| `-c, --credential` | 基本认证 |
| `--ssl` | 启用 SSL |
| `--ssl-cert` | SSL 证书 |
| `--ssl-key` | SSL 密钥 |
| `-t, --terminal-type` | 终端类型 |
| `-T, --terminal-title` | 终端标题 |

**Web 访问**:
- URL: `http://host:7681`
- 自动连接 WebSocket
- 全功能终端体验

## 数据流 / 控制流

```
浏览器访问 http://host:port
    ↓
HTTP Server 返回前端页面
    ↓
WebSocket 连接建立
    ↓
终端输入/输出流转
    ↓
PTY ↔ Shell/命令
    ↓
WebSocket 双向传输
```
