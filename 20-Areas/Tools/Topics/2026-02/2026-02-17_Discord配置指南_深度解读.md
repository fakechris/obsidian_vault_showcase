# Discord配置避坑指南 + Agent Terms实战

## 核心概念

Discord是OpenClaw的正确打开方式。
- 4区11频道，每个频道一个独立session
- 干净、隔离、不串味

---

## 第一步：创建Bot

1. 去 Discord Developer Portal 创建 Application
2. 在 Bot 页面拿 Token

---

## 三个必须打开的开关

- **Presence Intent**
- **Server Members Intent**
- **Message Content Intent**

⚠️ 少开一个，消息就收不到。没有报错，就是静悄悄地丢消息。

---

## 邀请链接

1. OAuth2 → URL Generator
2. Scopes 选 bot
3. Permissions 选 Administrator

**Administrator权限：**
- 最省事，一个权限搞定所有
- 风险：Bot能删你整个服务器
- 建议：自用没问题，多人服务器需要精细配置

---

## Agent Terms 配置

不用手写配置，直接告诉Agent：

> 「这是 Discord Bot Token：xxxxx。服务器 ID 是 xxxxx。帮我配好 Discord，只允许这个服务器用，私聊只接受我的消息，频道里不用 @ 你就能对话。」

一句话，Agent自己创建配置、重启服务、连接成功。

---

## 4区11频道示例

```
🏠 日常：#💬-日常 / #⚡-内观 / #🔨-践行
💰 增长：#💰-生财 / #💼-搬砖
✍️ 立言：#🐦-x / #📝-note / #📕-小红书
🤖 Orb 自治区：#📊-日报 / #🐦-提及
```

**每个频道 = 独立session**
- 在 #💰-生财 聊投资，不会污染 #🐦-x 的写作上下文

---

## 隐藏大杀器：频道Topic

频道描述会被 OpenClaw 当作这个频道的 system prompt。

例如：
- #⚡-内观 频道 topic 写了 "INTJ 导师模式"
- 进这个频道，Orb 自动变毒舌导师
- 出来又恢复正常
- 零配置，纯靠 topic

---

## Agent Terms 养成

刚出生的Agent是通用AI：
- 什么都答应
- 什么都很礼貌
- 什么观点都没有

**如何养成：**
1. 主动跟它吵架
2. 说「我的想法有漏洞你直接指出来」
3. 说「回复不要超过三段，废话太多」

这些对话，Agent自己会写入配置文件，形成专属风格。
