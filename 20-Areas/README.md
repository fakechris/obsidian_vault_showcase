## 🎯 与 Knowledge 层的区别

| 对比 | Knowledge 层 | Areas 层 |
|------|-------------|----------|
| **内容** | 抽象概念、理论 | 实际应用、实践 |
| **时间** | 永久价值 | 需要更新维护 |
| **链接** | 概念间连接 | 与项目连接 |
| **示例** | AI Agent 定义 | 具体 Agent 项目分析 |

**关系**: Knowledge 是**理论**，Areas 是**实践**。

---

## 🔄 工作流

### 从 Knowledge → Areas
- 学习新概念 → 在实际项目中应用 → 记录经验到 Areas

### 从 Areas → Knowledge
- 项目经验 → 提炼模式 → 更新概念笔记

### 从 Areas → Projects
- 领域研究 → 形成项目想法 → 在 Projects 层执行
