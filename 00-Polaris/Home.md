## 🔄 工作流展示

本项目展示了完整的 Pipeline 输出效果：

1. **输入** → Pinboard 书签 / 文章 / GitHub 项目
2. **解读** → LLM 生成 6 维度深度分析
3. **质检** → 自动质量评分
4. **提炼** → 提取 Evergreen 概念
5. **索引** → 更新 MOC 和双向链接
